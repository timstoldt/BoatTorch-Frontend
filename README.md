
# BoatTorch-Frontend

Welcome to the frontend repository for BoatTorch, a pontoon booking website designed for Torch Lake's premier pontoon rental company. This project is built using ReactJS and Bootstrap.

## About

BoatTorch is a comprehensive platform that simplifies the booking process for renting pontoons. This project aims to provide a seamless and user-friendly interface for customers to view available pontoons, select rental dates, and make reservations.

## Features

- **Responsive Design**: Ensures a seamless experience across various devices.
- **User Authentication**: Secure login and registration for customers.
- **Booking Management**: Easily view and manage pontoon reservations.
- **Real-time Availability**: Check pontoon availability in real-time.
- **Integrated Payment Gateway**: Secure and convenient payment options.

## Technologies Used

- **Frontend**: ReactJS, Bootstrap

## Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/timstoldt/BoatTorch-Frontend.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd BoatTorch-Frontend
   ```

3. **Install dependencies:**
   ```bash
   npm install
   ```

4. **Run the development server:**
   ```
   bash
   npm start
   ```


## Usage
- Open your browser and navigate to http://localhost:3000 to view the application.
- Sign up or log in to your account.
- Browse available pontoons, select your preferred dates, and make a booking.
